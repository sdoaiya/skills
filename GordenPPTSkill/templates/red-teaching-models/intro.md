# 高级红色 · 数智教学图解（图型合辑）

## 一句话
与"高级红色·教学实施框架"同系列的高信息密度红色教学 deck，30 页主打**丰富图型合辑**（漏斗 / 齿轮 / 鱼骨 / 金字塔 / 六边形放射 / 同心圆 / 铅笔信息图 / 雷达等），主题偏数智技术与智慧教育，适合按"图型 + 教学环节"单页挑选拼装的教师教学比赛、说课、课改汇报。

## 风格标签
- 整体风格：**党政红 / 数智教学 / 高信息密度 / 多样图型**
- 排版复杂度：**极高**（单页常 30-60 个文本位，图型种类比同系列更丰富）
- 视觉元素：倒三角漏斗、齿轮三联、鱼骨图、六边形放射、同心圆、铅笔造型信息图、雷达、环形循环、实拍照片
- 适合人群：职业院校 / 中小学教师、教研员、参赛选手、师范生、培训机构教师

## 主题色
- 主色：`#C00000`（党政深红）
- 强调：`#E8542A`（暖橙渐变）
- 点缀：`#9864B8`（紫色，部分页面）
- 底色：`#FFFFFF`（白底正文）

## 字体
- 中文标题/正文：思源黑体 CN（Bold / Medium）
- 部分大标题：思源宋体 CN Heavy
- 少量：微软雅黑
- 英文/数字：Arial / 主题拉丁字体

## 适用场景（不要被示例专业限制）
示例文字横跨数智技术、数控、FPGA/CPLD、护理叙事医学等专业，**均为占位示例**，请整体替换为你的学科。其图解版式适合任何"教学实施 / 教学设计 / 课改"展示：
- 教师教学能力大赛、教学创新大赛"教学实施报告"
- 数字化 / 智慧教育 / 产教融合主题汇报
- 说课、公开课、示范课教学设计
- 课程改革、人才培养方案、课程标准解读
- 教研课题汇报、校企合作育人成果
- 任何需要多种专业图型的教育类汇报

## 模板包含的页面结构（共 30 页，全部可用）
全部为内容页，无独立封面 / 目录 / 致谢页。**章节面包屑是打散的**（不是严格 1→30 线性），更适合按图型单页挑选：
- **漏斗 / 收敛类**：slide 2、7、9、10、20（实施路径、校企合作、评价指标、融合创新、平台资源）
- **齿轮 / 循环类**：slide 11、14、19、28、29、30（评价机制、策略设计、团队协同、教/学双向、任务驱动、课后拓展）
- **放射 / 网络类**：slide 4、12、13（教学目标脑图、前沿技术六边形、智能学习同心圆）
- **鱼骨 / 树状类**：slide 16、21（成效归因、巩固/复习/实践）
- **数据 / 雷达类**：slide 1、17、25（评估概览、达成数据、可视化雷达分析）
- **特色信息图**：slide 26（文化传播/教育技术/认知科学 铅笔造型）
- **环节面包屑页**：slide 14/15/16/17/20/22/23/24/25/28/29/30（带"教学整体设计·…"等环节标签）

## 不要选用的页面
- 无（没有模板宣传 / 稻壳页，30 页均为可用内容页）

## 注意事项
- **按图型选页**：本套优势是图型丰富，建议先想清楚"这一页要用什么图（漏斗 / 鱼骨 / 放射…）"再挑页填字。
- **信息密度极高**：`max_chars` 多为单词 / 短语级，用等长或更短的词替换，避免顶破方框；同级标题保持原字号。
- **图形不随文字联动**：漏斗层数、齿轮咬合、鱼骨支数、环的弧度都是固定的，改文字不会改图；需要增减条目请同步调整图形或换页（`shape_caution_pages` 标了全部 30 页）。
- **字体**：思源黑体 / 思源宋体是该模板的标志，缺字体会 fallback 到系统字体，视觉略打折但不影响编辑。
- 部分页面带紫色 `#9864B8` 强调色，与主红色搭配，替换文字时无需改色。
