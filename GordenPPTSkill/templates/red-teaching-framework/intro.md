# 高级红色 · 教学实施框架（综合教学设计）

## 一句话
红橙渐变 + 思源黑体的高信息密度教学图解 deck，30 页一条完整教学逻辑链（教学整体设计→教学实施过程→学习效果评价→反思与改进→教学内容图谱），适合教师教学能力比赛、说课、公开课、课程思政、教研课题汇报等需要"满版专业图"的场景。

## 风格标签
- 整体风格：**党政红 / 教学专业 / 高信息密度 / 图解型**
- 排版复杂度：**极高**（单页常 30-60 个文本位，几乎每页一张大信息图）
- 视觉元素：树状分支、环形循环、漏斗、金字塔/三角、鱼骨图、流程箭头、四宫格矩阵、数据柱状、实拍课堂照片
- 适合人群：职业院校 / 中小学教师、教研员、参赛选手、师范生、培训机构教师

## 主题色
- 主色：`#C00000`（党政深红）
- 强调：`#E8542A`（暖橙渐变）
- 点缀：`#FFC000`（金黄）
- 底色：`#FFFFFF`（白底正文）

## 字体
- 中文标题/正文：思源黑体 CN（Bold / Medium / Heavy）
- 少量：微软雅黑
- 英文/数字：Arial / 主题拉丁字体

## 适用场景（不要被示例专业限制）
模板里的示例文字横跨思政（三爱统一 / 爱国主义教育）、会计、数控等专业，**全部是占位示例**，请整体替换为你自己的学科。其图解版式适合任何"教学实施 / 教学设计"展示：
- 教师教学能力大赛、教学创新大赛的"教学实施报告"
- 说课 / 公开课 / 示范课的教学设计展示
- 课程思政、思政课一体化建设汇报
- 教研课题开题 / 中期 / 结题汇报
- 专业人才培养方案、课程标准解读
- 校本教研、集体备课成果展示
- 任何需要"满版红色专业图"的教育类汇报

## 模板包含的页面结构（共 30 页，全部可用）
全部为内容页，无独立封面 / 目录 / 致谢页，按教学逻辑链组织：
- **教学整体设计**（slide 1-4）：模块化教学体系、教学目标拆解、教学内容分析、教学策略设计
- **教学实施过程**（slide 5-7）：课前准备、课中实施、课后拓展
- **学习效果评价**（slide 8-9）：评价方式、数据分析（含得分率列表）
- **反思与改进**（slide 10-12）：教学成效、存在问题、动态反馈机制
- **教学模型 / 策略研究**（slide 13-21）：核心素养金字塔、区域推进保障、四维一体模型、教师能力进阶、5E 探究环、鱼骨研究法等
- **教学内容图谱**（slide 22-30）：四维分析框架、四核驱动、能力路径、PDCA 时间轴、三阶递进资源、岗课证融合、立体化教学空间、数字孪生等

## 不要选用的页面
- 无（没有模板宣传 / 稻壳页，30 页均为可用内容页）

## 注意事项
- **逻辑链固定**：建议保持"整体设计→实施→评价→反思→内容图谱"的环节顺序；左上角形如"教学整体设计 /  教学目标"是环节面包屑，改其中一段要与对应章节呼应。
- **信息密度极高**：每个 slot 的 `max_chars` 多为单词 / 短语级，务必用等长或更短的词替换，避免顶破方框；同级标题保持原字号。
- **图形不随文字联动**：树状 / 环形 / 漏斗 / 鱼骨 / 箭头都是固定形状，改文字不会改变箭头长度、格子数量、环的弧度；需要增减条目时请同步调整图形或换页面（`shape_caution_pages` 标了全部 30 页）。
- **字体**：思源黑体是该模板的标志，缺字体会 fallback 到系统黑体，视觉略打折但不影响编辑。
- 这是一套"图先行"的模板：先按你需要的图型 / 教学环节选页，再往里填字，效果最佳。
