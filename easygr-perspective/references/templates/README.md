# EasyGR 模板索引

## 模板列表

1. `01-government-report-template.md`  
   政府汇报版 / 政府购买服务方案 / 领导汇报压缩版

2. `02-regional-playbook-template.md`  
   区域起盘 / 产业带下沉 / 样板区打法

3. `03-sales-collaboration-template.md`  
   G端与销售协同 / 活动转化 / 线索承接机制

4. `04-project-retrospective-template.md`  
   重点项目复盘 / 活动复盘 / 试点复盘

## 使用顺序

1. 先用 `EasyGR` 判断问题本质；
2. 再选对应模板；
3. 最后按对象切换表达：
   - 对政府：去销售化
   - 对内部：强化责任、节奏、结果口径、沉淀与复制
