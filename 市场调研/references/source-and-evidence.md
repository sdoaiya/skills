# 数据源与证据分级

## 证据等级

| 等级 | 来源 | 用途 |
|---|---|---|
| P1 | 政府、海关、统计局、监管披露、国际组织原始数据库 | 核心规模、贸易流向、政策约束、宏观变量 |
| P2 | 行业协会、上市公司年报、招股书、交易所公告、官方国别指南 | 行业结构、企业经营、国别准入、竞争格局 |
| P3 | 可复核业务数据：招投标、项目库、价格、库存、渠道、采购平台 | 验证景气、订单、项目落地和需求方向 |
| P4 | 券商、咨询、商业数据库、专业媒体 | 方法补充、线索发现、口径参考 |
| P5 | 搜索指数、舆情、招聘、地图热力、社媒 | 辅助判断变化方向，不支撑核心结论 |

## 行业出海核心来源

### 全球贸易与关税

- UN Comtrade：HS 编码维度的全球贸易主数据。
- World Bank WITS：贸易流、关税、非关税壁垒交叉验证。
- ITC Trade Map：目标国进口来源结构、竞争国对比。
- USITC DataWeb：美国市场细项核验。
- 中国海关总署：中文口径下的中国出口数据。

### 中国官方国别资料

- 商务部或地方商务主管部门发布/转载的《国别贸易指南》。
- 《对外投资合作国别（地区）指南》。
- 商务部国别贸易统计与国别营商信息。
- 中国出口信用保险公司国家风险报告。

### 行业协会与企业

- 用户指定行业的中国、目标国、区域和国际行业协会。
- 用户指定行业的上市公司年报、招股书、交易所公告和投资者演示。
- 用户指定行业的龙头企业、竞争企业、渠道商、平台方、采购方公开披露。
- 若用户研究工程机械，可参考中国工程机械工业协会、CECE、AEM、VDMA、JCMMA，以及三一、徐工、中联、柳工、山推、Caterpillar、Komatsu、Hitachi、Volvo CE、Liebherr 等企业披露。

### 需求侧宏观与项目

- IMF WEO：GDP、投资、通胀、汇率、预测。
- World Bank Open Data：城市化、固定资本形成、人口、收入、消费、产业结构等。
- OECD、World Bank、IMF、WTO、UNIDO 等宏观、产业和贸易数据库。
- World Bank、ADB、AfDB、IDB、AIIB 等项目库，适用于项目型、基建型或政府采购型行业。
- 各国主管部门、监管机构、财政预算、采购平台、招投标平台和行业监管数据库。

## 数据登记表字段

每个来源必须记录：

- source_id
- 来源名称
- 发布机构
- 证据等级
- 年份/发布日期
- URL 或本地路径
- 数据口径
- 使用章节
- 可复核方式
- 限制与偏差

## 冲突处理

1. 优先核对统计口径：HS 版本、产品范围、币种、年份、贸易流向。
2. 再核对发布主体：原始统计优先于二次转述。
3. 再核对时间：最新数据优先，但不得混用不同年度口径。
4. 差异无法消除时，用区间值，并解释差异来源。
